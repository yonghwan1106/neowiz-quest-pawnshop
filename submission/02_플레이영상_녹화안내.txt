================================================================================
                    기억의 전당포 - 플레이 영상 녹화 안내
================================================================================

■ 영상 요구사항

- 길이: 10분 이내
- 해상도: 1920x1080 권장
- 형식: MP4 (H.264 코덱)

■ 녹화 권장 소프트웨어

- OBS Studio (무료): https://obsproject.com/
- Windows 게임 바 (Win + G)
- Nvidia ShadowPlay (GTX 그래픽카드 사용 시)

■ 녹화 시나리오 (약 8분 구성)

[0:00 ~ 0:30] 타이틀 화면
- 게임 시작 화면 표시
- "시작" 버튼 클릭

[0:30 ~ 1:30] 전당포 소개
- 전당포 인테리어 확인
- 평판 시스템 UI 확인
- "손님 기다리기" 클릭

[1:30 ~ 4:00] 손님 대화 (민지)
- 민지 캐릭터 등장
- 대화 진행 (전체 읽기)
- 기억 구슬 설명 장면

[4:00 ~ 6:00] 기억 퍼즐
- 퍼즐 시스템 설명
- 4개 파편을 슬롯에 배치
- 감정 완료 애니메이션

[6:00 ~ 7:00] 선택지 결정
- 3가지 선택지 표시
- 원하는 선택 수행
- 선택 결과 확인

[7:00 ~ 8:00] 엔딩 화면
- 엔딩 타입 표시
- 평판 수치 카운트업
- 엔딩 일러스트 확인

■ 녹화 팁

1. 게임 창을 전체 화면으로 설정 (F11)
2. 마우스 움직임은 천천히
3. 대화 텍스트는 읽을 시간 확보
4. BGM이 재생되지 않을 수 있음 (오디오 파일 미포함)

■ 파일 이름

녹화 완료 후 파일명: 02_플레이영상_기억의전당포.mp4
