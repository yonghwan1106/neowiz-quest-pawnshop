================================================================================
                    기억의 전당포 (The Pawnshop of Memories)
                         프로토타입 실행 안내서
================================================================================

■ 실행 방법

1. Godot Engine 4.2.x 버전을 설치합니다
   - 다운로드: https://godotengine.org/download/archive/4.2.2-stable/

2. 03_실행빌드_기억의전당포.zip 파일의 압축을 해제합니다

3. Godot Engine을 실행하고 "Import" 버튼을 클릭합니다

4. 압축 해제한 폴더에서 "project.godot" 파일을 선택합니다

5. 프로젝트가 열리면 F5 키를 눌러 게임을 실행합니다


■ 조작법

- 마우스 클릭: 대화 진행, 버튼 선택
- 드래그 앤 드롭: 기억 퍼즐 파편 이동


■ 게임 플로우

1. 타이틀 화면 → "시작" 버튼 클릭
2. 전당포 화면 → "손님 기다리기" 버튼 클릭
3. 손님과 대화 진행 (클릭으로 진행)
4. 기억 퍼즐 → 파편을 슬롯에 배치
5. 선택지 선택 (매입/거절/수정)
6. "다음 날" 버튼 → 엔딩 화면


■ 엔딩 확인 방법

평판에 따라 3가지 엔딩이 표시됩니다:
- 자비로운 전당포지기 (자비 점수 최고)
- 정의로운 전당포지기 (정의 점수 최고)
- 냉철한 전당포지기 (이익 점수 최고)
